# Project Roadmap - SignalSense

## Overview
SignalSense is a communication pattern analysis tool designed to track response times, consistency, and interaction frequency over time. The goal is to help users better understand communication behavior using data instead of assumptions.

## Sprint 1 Goals
- Create GitHub repository
- Write project idea and README
- Research similar tools or concepts
- Set up project structure

## Sprint 2 Goals
- Build basic input system for logging communication data
- Store data using file handling
- Begin developing functions for tracking response time

## Sprint 3 Goals
- Analyze stored data
- Calculate patterns (frequency, delays, consistency)
- Display results clearly to the user

## Sprint 4 Goals
- Improve user experience
- Add summaries or simple insights
- Test program and fix errors

## Notes
This project will use Python concepts learned in class, such as functions, conditionals, and file handling to store and analyze communication data. The goal is to turn communication patterns into something clear and measurable so users can better understand real behavior instead of relying only on assumptions or emotions.
